## 📂 Repository Structure

```text
OBJECT-ORIENTED-ANALYSIS-AND-DESIGN-Learning-Materials/
│
├── 📁 MID/        # Mid-term related notes, slides & materials
├── 📁 FINAL/      # Final exam related notes, slides & materials
└── 📁 EXTRA/      # Additional resources & supplementary content
```

---

## 📖 Topics Covered

- ✅ Object-Oriented Concepts (Class, Object, Inheritance, Polymorphism, Encapsulation, Abstraction)
- ✅ UML Diagrams (Use Case, Class, Sequence, Activity, State, Component, Deployment)
- ✅ Design Patterns (Creational, Structural, Behavioral)
- ✅ SOLID Principles
- ✅ System Analysis & Modeling
- ✅ Software Development Life Cycle (SDLC)
- ✅ Case Studies & Practice Problems

---

## 🎯 Purpose

This repository is intended to help students:

- Prepare for **mid-term** and **final** examinations
- Understand core OOAD concepts through organized notes
- Practice with extra materials and reference resources

   ```bash
   git clone https://github.com/shafikulnayem10/OBJECT-ORIENTED-ANALYSIS-AND-DESIGN-Learning-Materials.git
   ```

   ```bash
   cd OBJECT-ORIENTED-ANALYSIS-AND-DESIGN-Learning-Materials/MID
   ```
